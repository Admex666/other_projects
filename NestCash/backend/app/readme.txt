# NestCash API

This is the main entry point of the NestCash API.

## Overview

The API is built using FastAPI and provides endpoints for authentication and data management.

## Structure

The API is organized into the following folders:

* `core`: Contains the core functionality of the API, including database initialization and security settings.
* `routes`: Holds the API routes, including authentication and data endpoints.
* `services`: Contains the services used by the API, including authentication and data management.
* `models`: Defines the data models used by the API.
