# NestCash App

This is the main entry point of the NestCash app.

## Overview

The app is built using Flutter and provides a basic authentication flow.

## Structure

The app is organized into the following folders:

* `screens`: Contains the app's screens, including authentication and dashboard.
* `services`: Holds the authentication service.
* `widgets`: Reusable UI components.

## Getting Started

To run the app, navigate to the project root and execute `flutter run`.
